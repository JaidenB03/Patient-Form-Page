Overview:

This project is a patient registration form created as part of a web development course assignment. The website simulates an intake form for a fictional medical office called Aquamarine Medical and focuses on handling user data, validating input, and working with browser storage.

Features:

-Dynamic date display using JavaScript
-Form validation for email, ZIP code, password, and date of birth
-"Remember Me" functionality using cookies
-Local storage for saving form data
-Personalized welcome message for returning users
-Dynamic state dropdown populated using Fetch API from a JSON file
-Embedded iframe with external content
-Sticky header and fixed footer for improved layout

Technologies Used:

HTML
CSS
JavaScript
JSON
Fetch API
Cookies and Local Storage

Known Issues:

-The "Review" button is still being debugged
-The Contact Us link is not yet connected to an active page

What I Learned:

Through this project, I learned how to work with browser storage (cookies and local storage), dynamically load content using Fetch API, apply form validation, and improve user experience using layout techniques such as sticky headers and iframes.
